## 🚀 GitHub Pagesデプロイ方法

### 基本手順
1. GitHubリポジトリを作成
2. ファイルをプッシュ
3. Settings > Pages で設定
4. Source: "Deploy from a branch"
5. Branch: "main", Folder: "/ (root)"

詳細な手順は `DEPLOYMENT.md` を参照してください。
